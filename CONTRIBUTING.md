Contributing and Acceptable Use

This repository is a tutorial and sample code for development and testing purposes. By contributing or using this repository you agree to the following:

1. No redistributing third-party binaries
   - Do not add or commit third-party installers, binaries, or other redistributable assets (for example, Fiddler Classic or Bruno installers) unless you have explicit permission to redistribute them and include the appropriate license and attribution.

2. No sensitive data
   - Do not commit secrets, API keys, credentials, proprietary company code, internal hostnames, or other sensitive information. If you accidentally commit such data, remove it and rotate any exposed credentials immediately.

3. Respect third-party licenses and trademarks
   - Contributions should not claim ownership of third-party tools. If you add instructions or scripts that interact with external tools, include links to their official documentation and respect their license/usage terms.

4. Code of conduct
   - Keep contributions professional and relevant to the tutorial's purpose. The repository maintainer reserves the right to reject or remove content that is inappropriate or violates third-party licenses.

5. Contributor license
   - By submitting a pull request you confirm that you have the right to contribute the changes and that those changes are eligible to be published under the repository's MIT license.

If you need help or want changes applied, open an issue or a pull request. For legal questions about licensing, trademarks, or redistribution rights consult your legal team.
